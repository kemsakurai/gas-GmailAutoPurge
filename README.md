# このアプリケーションについて

このアプリケーションは、Gmailの検索クエリに合致するメールをGmailの受信ボックスから削除する    
Googleスプレッドシートと、Google Apps Scriptで構成されたWebアプリケーションです。       

## インストール、デプロイ      

* **Install & Deploy**   
```console
git clone https://github.com/kemsakurai/gas-GmailAutoPurge.git
npm install -g clasp
npm install
sh build.sh && clasp push
```
